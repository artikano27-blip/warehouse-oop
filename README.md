# 📦 Warehouse Management System (Ядро системы складского учета)

## Описание проекта
Проект представляет собой масштабируемое ядро системы управления складом (WMS), разработанное на языке Python. Архитектура построена на фундаментальных принципах объектно-ориентированного программирования (ООП). 

Система предназначена для строгого учета складской номенклатуры, контроля остатков, автоматического расчета стоимости активов и управления логистикой (подготовкой к отгрузке) в зависимости от специфики груза. В коде реализована надежная защита данных (Data Validation), исключающая появление логических аномалий (например, отрицательных цен или минусовых остатков товара).

## 🛠 Реализованный функционал
* **Иерархия товаров:** Поддержка различных типов номенклатуры (стандартные грузы, хрупкие товары со шкалой хрупкости, скоропортящиеся продукты с таймером годности).
* **Умная логистика:** Система автоматически определяет правила упаковки и транспортировки для каждого конкретного типа товара.
* **Агрегация данных:** Единый центр управления (Склад), позволяющий добавлять новые объекты, отслеживать инвентарь и рассчитывать общую капитализацию в режиме реального времени.
